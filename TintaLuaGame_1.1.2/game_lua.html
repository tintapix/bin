game.addaction

Добавляет действие такое как обработка контролов или таймер

Param: Строка тип действия один из вариантов:

"key_down"

"key_up"

"timer"

для кнопок - номер кнопки клавиатуры или мыши

при обработки которой будет выполнено действие

Очищает сцену от объектов - удалает все группы вместе с объектами



game.setparent

Задает родительский объект для другого объекта в одной группе. Поворот и перемещения для дочернего объекта становятся

зависимыми от родительского

Param: Строка имя группы

Param: Целое порядковый номер дочернего объекта в группе начиная с 0

Param: Целое порядковый номер родительского объекта в группе начиная с 0



game.setmaterial

Задает материал для объекта

Param: Строка имя группы

Param: Целое порядковый номер

Param: Строка имя или путь материалу



game.materialparam

Задает параметр в материале для объекта

game.materialparam("demo",0,"fill.material",color, 0.1, 0.3, 0.5)

game.materialparam("demo",0,"fill.material",angle, 0.1 )

Param: Строка имя группы

Param: Целое порядковый номер

Param: Строка имя материала

Param: Строка имя параметра

Param: значение параметра в зависимости от параметра(для составных параметров значение разбиваетсмя запятыми таких как вектор или матрица)



game.addobject

Добавляет объект в группу может быть 2-х вариантов: game.addobject("Group", 10.0, 10.0, 50.0, "texture.png" )

game.addobject("Group", 10.0, 10.0, 50.0, 0,255,0 ), если изображение или цвет н езаданы то объект становится невидимым(showobject с параметром false)

Param: Строка имя группы

Param: Число с плавающей точкой координата x

Param: Число с плавающей точкой координата y

Param: Число с плавающей точкой габаритный размер(0 можно установить, что бы объект не рисовался, но прискутствовал в сцене)

Param: Строка или целое путь к изображению c разрешением .png, .jpg или .jpeg, или материалу с разрешением .material или компонент цвета r

Param: Целое компонент цвета g

Param: Целое компонент цвета b

Повернуть объект на угол, если угол < 0 то против часовой, иначе по часовой

Param: Целое порядковый номер объекта в группе начиная с 0

Param: Число угол в градусах от -360 до 360

Param: Необязательный Число с плавающей точкой скорость перемещения градусов в миллисекунду

Param: Необязательный Строка команда для выполнения после завершения

Param: Необязательный Строка команда для выполнения после завершения



game.angle

Возвращает угол между вектором вперед и вектором к точке для объекта по ближайшей дуге.

Если точка левее оси объекта угол положительный, если правее то отрицательный

Param: Строка группа

Param: Целое порядковый номер объекта в группе начиная с 0

Param: Число координата точки второго вектора x

Param: Число координата точки второго вектора y

Return: Число с плавающей точкой угол в градусах

Param: Необязательный Строка команда для выполнения после завершения

Смещает объект на значение относительно координат объекта, вперед "+" по x, вправо "+" по y,

Param: Строка имя группы

Param: Целое порядковый номер объекта в группе начиная с 0

Param: Число с плавающей точкой x

Param: Число с плавающей точкой y

Param: Необязательный Строка команда для выполнения после завершения

Ищет препядствие на пути объекта в направлении вперед, среди объектов указанной группы.

Если препядствия не нашлось, возвращает nil. local index = game.getfore( "player", 0, "stones", 500 )

Return: Целое индекс ближайшего элеиента из группы или nil, если пересечения нет

Return: Расстояние до центра или nil, если пересечения нет

game.isplaying

Возвращает признак проигрывания файла. Удобно, когда playsound приходится вызывать в циклах. Звук тогда воспроизводится только вначале, а проверив isplaying можно не вызывать playsound пока не воспроизведется до конца

if game.ispalying("звуковой_файл") == false then

game.playsound("звуковой_файл")

end

Return: Булевое true - файл проигрывается в данный момент
